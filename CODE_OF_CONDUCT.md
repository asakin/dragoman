# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct. The full, canonical text lives at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful in issues, PRs, and commits; assume good faith; keep critique focused on the work, not the person. The full document covers expectations, scope, and the enforcement framework.

## Reporting

If you experience or witness behavior that violates the Covenant, contact the maintainer directly:

- **Maintainer:** Ariel Sakin
- **Email:** asakin@gmail.com

Reports are handled confidentially. The maintainer will respond within a reasonable window and act according to the enforcement guidelines in the Covenant.

## Scope

This Code of Conduct applies in all project spaces — issues, PRs, commits, discussions, and any related public channels — and when an individual is officially representing the project.
